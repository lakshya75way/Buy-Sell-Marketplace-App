## How to get started

1.  **Check Flutter**: Make sure you have Flutter installed on your computer.
2.  **Get the Code**: Clone this project and open it in your favorite editor.
3.  **Install Plugins**: Run `flutter pub get` in your terminal to get all the necessary pieces.
4.  **Launch**: Plug in your phone or start an emulator, then hit `flutter run`.

_Quick Tip: If you're using an emulator and want to try the biometric login, remember to "enroll" a fingerprint or face in the emulator's Android settings first!_
